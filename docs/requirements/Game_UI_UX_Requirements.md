## 1. 产品概述

### 1.1 产品定位

一款 **iOS 悬疑叙事游戏**，采用 **16-bit 像素艺术风格**，通过 **语音交互** 与 NPC 对话，体验随机生成的悬疑故事。

### 1.2 核心价值主张

| 价值点 | 描述 |
|--------|------|
| **沉浸式叙事** | 每次游玩都是独特的悬疑故事体验 |
| **自然交互** | 语音输入让玩家自然融入对话情境 |
| **复古美学** | 16-bit 像素风格唤起经典游戏情怀 |
| **高重玩性** | 随机生成故事 + 多结局设计 |

### 1.3 产品名称（暂定）

**Mystery Files** / **悬疑档案**

## 2. 目标用户

### 2.1 用户画像

| 属性 | 描述 |
|------|------|
| **年龄** | 18-35 岁 |
| **游戏偏好** | 悬疑/推理/叙事驱动游戏 |
| **设备** | iPhone 用户（优先），后续支持 iPad |
| **语言** | 中文/英文双语用户 |

### 2.2 使用场景

- 通勤时间的沉浸体验（每局 30 分钟+）
- 睡前放松游戏
- 喜欢剧情解谜的玩家

## 3. 核心玩法

### 3.1 游戏循环

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  开始新案件  │ ──→ │  场景探索   │ ──→ │  NPC 对话   │
└─────────────┘     └─────────────┘     └─────────────┘
       ↑                                       │
       │                                       ↓
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  开始新案件  │ ←── │  查看档案   │ ←── │  结局问答   │
└─────────────┘     └─────────────┘     └─────────────┘
```

### 3.2 故事生成机制

| 组件 | 实现方式 |
|------|----------|
| **故事框架** | AI 实时生成 |
| **角色设定** | AI 生成 NPC 背景、动机 |
| **场景设计** | AI 生成可探索场景 |
| **线索分布** | AI 分配关键线索位置 |
| **结局分支** | 根据玩家表现动态生成 |

### 3.3 问答环节

结局后系统会提出 2-3 个问题，玩家有 2 次提问机会解答故事疑问。

## 4. 视觉设计规范

### 4.1 整体风格

| 属性 | 值 |
|------|-----|
| **风格定位** | 极简影院风格 |
| **背景主色** | 纯黑 `#000000` 或 深灰渐变 `#0A0A0A → #1A1A1A` |
| **强调色** | 青色 `#2EC4B6` |
| **文字颜色** | 白色 `#FFFFFF` / 浅灰 `#E0E0E0` |
| **次要文字** | `#8E8E93` |

### 4.2 16-bit 像素艺术规范

| 元素 | 规范 |
|------|------|
| **色彩深度** | 16-64 色 |
| **像素粒度** | 清晰可见，不开启抗锯齿 |
| **角色立绘** | 全身/半身像，支持表情切换 |
| **场景图** | 16:9 比例，居中显示 |

### 4.3 字体规范

| 用途 | 规范 |
|------|------|
| **主要字体** | 像素字体优先，或等宽字体 |
| **标题字号** | 24px - 32px |
| **正文字号** | 14px - 18px |
| **辅助文字** | 12px |

### 4.4 圆角与间距

| 元素 | 值 |
|------|-----|
| **卡片圆角** | 0px（像素风格，无圆角）或 4px |
| **按钮圆角** | 4px - 8px |
| **对话框边距** | 16px - 24px |
| **元素间距** | 8px 网格系统 |
