# Audio Resources

Place your audio files in these directories:

## BGM (Background Music)
- `bgm/` - Background music tracks (.ogg format recommended)

Required tracks:
- `main_theme.ogg` - Main menu theme
- `mystery_1.ogg` - Exploration scene BGM
- `tension.ogg` - Dialogue/tension BGM
- `ending.ogg` - Ending theme

## SFX (Sound Effects)
- `sfx/` - Sound effects (.wav format recommended)

Required SFX:
- `button_click.wav` - Button click sound
- `dialogue_type.wav` - Typewriter effect sound
- `clue_collect.wav` - Clue collection sound
- `transition.wav` - Scene transition sound

## Ambient
- `ambient/` - Ambient sounds (.ogg format recommended)

Required ambient:
- `rain.ogg` - Rain sound
- `clock_ticking.ogg` - Clock ticking sound
- `wind.ogg` - Wind sound

## Audio Specifications

| Type | Format | Sample Rate | Channels |
|------|--------|-------------|----------|
| BGM | OGG Vorbis | 44.1kHz | Stereo |
| SFX | WAV | 44.1kHz | Mono/Stereo |
| Ambient | OGG Vorbis | 44.1kHz | Stereo |

## Volume Levels

Default volume settings (in dB):
- BGM: 0.0 dB
- SFX: 0.0 dB
- Ambient: -5.0 dB
- Ducking (during dialogue): -10.0 dB
